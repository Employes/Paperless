# Input

## Usage:

```html
<input class="p-input" ...props />
```

## Support

Current we only support a small list of types:

-   text
-   number
-   date
-   month
-   datetime-local
-   time
-   tel
-   password
-   email
-   textarea
-   checkbox
-   toggle
-   radio
